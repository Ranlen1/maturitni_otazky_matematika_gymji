# maturitni-otazky-matematika-gymji
Tento repozitář byl vytvořen Radimem Guichenem a Reném Čakanem a má sloužit jako teoretický podklad ke všem maturitním otázkám. Pro plné pochopení otázky ovšem doporučujeme si vypočítat alespoň pár příkladů z materiálů poskytnutých učiteli či z učebnic, jako je například Petáková. Pro nejpohodlnější čtení doporučujeme repozitář naklonovat a nebo si ho stáhnout jako zip. Pokud klikneš na zelené tlačítko ,,code'', zobrazí se možnost ,,Download ZIP''. Pokud bys našel v textu nějaké chyby, neváhej zažádat o pull request a nebo mi napiš na ranlen@proton.me a mohu chyby opravit nebo poskytnout roli kolaborátora, čímž můžeš tento repozitář updatovat sám. Pokud tento repozitář využiješ, velice bychom ocenili, kdybys o něm dal vědět mladším ročníkům, ať nezmizí v hlubinách internetu. Předem děkujeme a hodně štěstí s maturitou.
